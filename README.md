# Agentic Note

Quick single-note workspace built with Next.js and React. Content stays in your browser via `localStorage`, so you can jot something down and come back to it later without worrying about accounts or syncing.

## Getting Started

- `npm install`
- `npm run dev`
- `npm run build`
- `npm start`
- `npm run lint`

The app renders at `http://localhost:3000`.

## Features

- Persistent storage in the browser (no backend needed)
- Title and rich text area with live word and character counts
- One-click copy and clear actions
- Responsive glassmorphism UI with light and dark themes
